# ft_services

## Overview
Kubernetesを用いて７つのサービスをデプロイする。
alpineイメージから作成する制約あり。

## Requirement
- XUbuntu 18.04.5
- minikube v1.14.2
- kubectl v1.18.0
- docker v19.03.6

## Usage
```
git clone ...
cd ft_services
./setup.sh
```
